# Roadmap

## Phase 0: Protocol Sketch

- Define `llm-ir.v0` JSON Schema.
- Publish representative examples.
- Build a small CLI prototype.
- Document privacy and safety boundaries.

## Phase 1: LLM Adapter

- Add encoder and decoder interfaces.
- Support OpenAI-compatible and local model backends.
- Add test fixtures for meaning preservation.
- Add basic schema validation.

## Phase 2: Evaluation

- Create a benchmark for multilingual meaning preservation.
- Measure source-language marker reduction.
- Compare direct translation with IR-mediated generation.
- Add human review workflows.

## Phase 3: Applications

- Browser extension demo for social posts.
- API gateway for apps that need multilingual output.
- SDKs for TypeScript and Python.
- Community examples for more languages.

## Phase 4: Governance

- Stabilize schema versioning.
- Define compatibility rules.
- Add safety review process for privacy transforms.
- Create an open working group for language and culture contributors.
